# Zen Smart Selection Extension

A lightweight, privacy-focused Manifest V3 extension designed exclusively for **Zen Browser**. 

This project bridges the gap between different browser ecosystems by bringing **Opera GX's beloved native text-selection features**—instant web search and automatic local currency conversion—directly into Zen Browser's minimalist environment.

## 🚀 Features
* **Smart Currency Detection:** Instantly converts foreign currencies (USD, EUR, GBP, etc.) to your local currency based on your system's locale without sending data to any external server.
* **Instant Search Popup:** Highlight any text and perform quick searches with zero distraction.
* **Ultra-Lightweight:** Written in pure, modern JavaScript with no heavy dependencies or bundlers, ensuring absolute performance and low RAM usage.
